# 版本列表源切换功能实现方案

## 1. 设计概述

基于当前代码库的结构，我将设计一个可扩展的下载源工厂系统，实现版本列表源的灵活切换。该系统将采用工厂模式，支持不同类型资源（Minecraft版本、ModLoader等）的下载源切换。

## 2. 核心组件设计

### 2.1 下载源接口定义

创建统一的下载源接口，定义资源获取的标准方法：

```csharp
public interface IDownloadSource
{
    string Name { get; }
    string GetVersionManifestUrl();
    // 可扩展的方法，用于其他资源类型
}
```

### 2.2 下载源实现

实现两种版本列表源：

```csharp
// 官方源实现
public class OfficialDownloadSource : IDownloadSource
{
    public string Name => "Official";
    public string GetVersionManifestUrl() => "https://piston-meta.mojang.com/mc/game/version_manifest.json";
}

// BMCLAPI源实现
public class BmclapiDownloadSource : IDownloadSource
{
    public string Name => "BMCLAPI";
    public string GetVersionManifestUrl() => "https://bmclapi2.bangbang93.com/mc/game/version_manifest.json";
}
```

### 2.3 下载源工厂

设计下载源工厂，用于创建和管理不同类型的下载源：

```csharp
public class DownloadSourceFactory
{
    private readonly Dictionary<string, IDownloadSource> _sources = new();
    
    public DownloadSourceFactory()
    {
        // 注册默认下载源
        RegisterSource("official", new OfficialDownloadSource());
        RegisterSource("bmclapi", new BmclapiDownloadSource());
    }
    
    public void RegisterSource(string key, IDownloadSource source)
    {
        _sources[key] = source;
    }
    
    public IDownloadSource GetSource(string key)
    {
        return _sources.TryGetValue(key, out var source) ? source : _sources["official"];
    }
}
```
