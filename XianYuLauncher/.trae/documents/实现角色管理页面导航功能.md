# 角色管理页面导航功能实现计划

## 1. 项目导航机制分析

### 1.1 导航服务实现

* 使用 `NavigationService` 类实现 `INavigationService` 接口

* 通过 `_navigationService.NavigateTo(typeof(ViewModel).FullName!, parameter)` 进行页面导航

* 页面类型与 ViewModel 类型关联，通过 `IPageService` 解析

### 1.2 现有页面导航示例

```csharp
// ModPage.xaml.cs
_navigationService.NavigateTo(typeof(ModDownloadDetailViewModel).FullName!, mod.ProjectId);

// 版本列表Page.xaml.cs
_navigationService.NavigateTo(typeof(版本管理ViewModel).FullName!, version);
```

```csharp
// 修改 ProfileCard_Tapped 方法
private void ProfileCard_Tapped(object sender, TappedRoutedEventArgs e)
{
    if (sender is Border border && border.Tag is MinecraftProfile profile)
    {
        // 导航到角色管理页面
        _navigationService.NavigateTo(typeof(角色管理ViewModel).FullName!, profile);
    }
}
```
